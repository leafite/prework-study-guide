HTML

  • The head element contains information about the webpage.
  • meta holds information such as char set, page description, keywords, author, and viewport settings.
  • main holds the majority/main context of the document
  • h1-6 determines the size of header printed 1 being the largest and often used for titles
  • section is used to keep together related context
  • ul/il/ol - these are used for listing. ul/ol defines if bullet or numerical listing while il is for each item listed.
  • img is used to display images and scr to locate the image being printed
  • footer typically contains information like author, copyright, contact, sitemap, and navigation.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • Declarations hold two components being {property:value}
  • If adding same styles to more than one HTML selector add a , (applies to both)
  • File is used to form connections with external files
  • A period followed by a word indicates a class selector (i.e. .card) in which if class is assigned to an HTML element then it will follow given properties
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git add -A ensures that all files are committed in the working branch (make sure in proper branch)
  • git commit -m "text" is used to write a short description of changes
  • git pull origin main is important as it makes sure your local branch is up to date with the remote project in GitHub
  • git push origin branch pushes all the changes unto GitHub
  • PR is used to merge the pushed branch into main
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Function declaration is done by function name(){ //code block}
  • Console.log is used to display information in console
  • Const is used to declare a variable
  • Control flow is the order in which a computer executes code in a script.
